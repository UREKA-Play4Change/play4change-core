# Contributing to U!REKA Play4Change

Thank you for your interest in the U!REKA Play4Change project! 

## Current Project Status: Closed to External Contributions
This project is currently an active academic assignment for the Unidade Curricular de Projecto e Seminário at the Instituto Superior de Engenharia de Lisboa (ISEL).  Because this is an evaluated project with a final delivery date of July 11, 2026, we cannot accept any external code contributions, pull requests, or bug fixes from the public at this time. 

Accepting outside code would violate academic integrity policies.

## Note for Recruiters and Reviewers
If you are viewing this repository as part of an internship application or portfolio review, welcome! While the repository is strictly maintained by our core student group, we enforce professional software engineering standards internally. 

Here is a brief overview of our internal team workflow:
* **Branching:** We use a standard feature-branch workflow (e.g., `feature/jwt-auth`, `bugfix/daily-reminders`).
* **Commits:** We write clear, descriptive commit messages to maintain a transparent record of individual contributions throughout the project.
* **Pull Requests:** All code must be submitted via a Pull Request using our standard PR template and reviewed before merging into the main branch.

## Future Plans
After our final milestone in July 2026, we will re-evaluate our licensing and open-source contribution model. At that time, we may open the project to community contributions regarding new configurable subject domains, audience profiles, or additional multiplatform targets.
